# WeChatRSS

Welcome to the WeChatRSS documentation. WeChatRSS is a lightweight, self-hosted solution for aggregating WeChat Official Account content and delivering it via secure, private RSS feeds.

## Core Value Proposition

*   **Privacy:** No third-party services involved. Your subscriptions are yours.
*   **Safety:** Uses advanced browser automation and stealth techniques to avoid WeChat's aggressive anti-bot measures.
*   **Full Text:** Automatically extracts full article content and mirrors images locally.
*   **Multi-User:** Dashboard for managing multiple users and their private feeds.

## Quick Start

1.  **Install:** `pip install -r requirements.txt`
2.  **Auth:** `python auth.py` (Scan QR code)
3.  **Run:** `python main.py`

Explore the navigation tabs to learn more about the architecture and features.
